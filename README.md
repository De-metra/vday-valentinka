# 💝 Интерактивная валентинка

## 📖 Описание 
Этот проект — интерактивная валентинка, созданная с любовью ❤️ для особенного человека на 14 февраля.

### 🛠️ Технологический стек
**Backend**

```bash
FastAPI     # Современный, быстрый фреймворк
Uvicorn     # ASGI-сервер для запуска
Python 3.8+ # Основной язык программирования
```

**Frontend**

```bash
HTML5       # Структура страниц
CSS3        # Стили и анимации
JavaScript  # Интерактивность
```

_💢 Примечание: весь фронтенд был полностью завайбкоден 🥸_

```text
vday-app/
├── app.py
```

#### 🚀 Быстрый старт
1. Установка зависимостей
```bash
pip install fastapi uvicorn
```
2. Запуск приложения
```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```
3. Открыть в браузере
```text
http://localhost:8000
```
